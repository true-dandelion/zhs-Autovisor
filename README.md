# zhs-Autovisor - 智慧树自动刷课助手

## 项目简介

zhs-Autovisor 是一个专为智慧树(Zhihuishu)在线学习平台设计的自动化刷课工具。它通过模拟真实用户行为，自动完成视频观看、答题、进度跟踪等学习任务，帮助用户高效完成在线课程学习要求。

## 主要功能

###  核心功能
- **自动视频播放**：自动播放课程视频，支持倍速播放(最高1.8倍)
- **进度监控**：实时监控学习进度，自动切换到下一个视频
- **智能答题**：自动完成课程中的弹窗题目，支持AI随堂练习
- **验证码处理**：自动处理登录时的滑块验证码
- **多课程支持**：支持同时处理多个课程链接

###  高级功能
- **浏览器自动化**：基于Playwright实现真实的浏览器操作
- **隐藏模式**：可选择隐藏浏览器窗口，后台运行
- **静音播放**：支持静音模式，不打扰其他工作
- **Cookie保存**：保存登录凭证，下次自动登录
- **进度保存**：实时保存学习进度，支持断点续学


## 使用方法

### 1. 配置设置
编辑 `configs.ini` 文件，设置以下参数：
```ini

[course-option]
limitSpeed = 1.5          # 播放倍速
soundOff = True           # 是否静音

[script-option]
enableAutoCaptcha = True  # 自动处理验证码
enableHideWindow = False  # 是否隐藏浏览器
```

### 2.1. 添加课程链接1
在配置文件中添加智慧树课程链接：
```ini
[course-url]
URL1 = https://studyvideoh5.zhihuishu.com/video/...
URL2 = https://studyvideoh5.zhihuishu.com/video/...
```

## 特色功能

###  智能学习流程
1. 自动登录智慧树平台
2. 加载课程页面并优化界面
3. 识别未完成的课程内容
4. 自动播放视频并监控进度
5. 处理弹窗题目和AI练习
6. 完成后自动切换到下一课程

###  安全特性
- **浏览器反检测**：使用stealth.js脚本减少被平台检测的风险
- **真实用户模拟**：模拟真实用户操作行为，避免异常检测
- **密钥验证**：内置密钥验证系统，确保合法使用

###  进度管理
- 实时显示学习进度百分比
- 支持断点续学功能
- 自动保存学习状态
- 详细的日志记录系统
